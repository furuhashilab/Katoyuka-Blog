### 第1章　はじめに<a name="はじめに"></a>

本研究は、TEDxAoyamaGakuinU 2018というソーシャルイベントの一つにおけるレセプション/アフターパーティでの孤立問題を改善する取り組みとして、イベントの中に様々試作を盛り込んだ方法と結果、その考察である。TEDxに限らず、不特定多数が交流するソーシャルイベント後のアフターパーティでは初対面同士の会話がしづらく、孤立してしまう通称「レセプション難民」現象が多々発生している。せっかく足を運んでいただいた人々が少しでも有意義な時間を過ごしていただくために、この「レセプション難民」問題を減らすいくつかのアイディアの効果検証を今回のTEDxAoyamaGakuinUでは実施した。

背景として、様々なスピーチ動画を配信する非営利団体TEDはよく知られた名前である。そこからまた独立した団体として、各地域コミュニティにTEDx団体が存在する。近年このTEDxイベントを行う団体は数を増し、1番始めに行われたTEDxTokyoから日本国内だけで400近くにまで増えた。

その中で私は自身の所属する青山学院大学でTEDxAoyamaGakuinUとしてイベントを開催することを決定した。イベント開催のために様々な準備をしてきた中で、他団体のTEDxの運営を手伝うために、また単純に観覧するためにオーディエンスとして参加することで自分の中のTEDxの知見を高めようとした。
参加することで感じた課題がTEDx運営上のレセプション会場の孤立問題である。自分自身がレセプションに参加することで初対面の人との会話が困難であり、また他者との会話の中で同じように会話にハードルを感じる人が存在するとわかった。

このような想いから自身が主催するTEDxAoyamaGakuinUではこうした「レセプション難民」を減少させようと取り組んだ。

また、この取り組みの中で、数多く存在する他のTEDxとの差異を出すために、またレセプションの参加者をより楽しませるために、策を考え実行し考察した。
この研究が来年以降のイベントでのレセプションへ貢献し、TEDxAoyamaGakuinUがより広く大勢の人々に楽しんでもらえるようなイベントになることを期待する。

本論文の以下の構成は次のようになっている。
 
 第２章では、本論文で使用する諸概念について述べる。
 
 第３章では、レセプション時に用いた検証施策について述べる。
 
 第４章では、それぞれの施策の検証結果、考察を述べる。
 
 最後に、第５章で本論文の結論を述べる。

### 第2章　テーマ概論説明
#### 第2章　1.TEDとは<a name="1.TEDとは"></a>

TEDとは -Idea worth spreading- とあるように、価値有るアイディアを世に広めることを目的として非営利団体である。1984年に始まった当初、TEDの名前のそれぞれの頭文字の通り、T... Technology (テクノロジー) E… Entertainment (エンターテイメント) D… Design (デザイン)に集中したおよそ18分前後にわたる短く、かつ印象的な動画を配信していた。今日ではジャンルに関係なく科学やビジネス、国際問題など様々なトピックに関するスピーチ動画を配信している。

現在の動画数は2900以上、公開されている動画の言語数は100以上に上る。
そしてこの団体は国際的コミュニティであり、価値あるアイディアを世に広めるという目的においてどんな人にも広く公開されている。本場の講演会にスピーカーとして参加することも可能である。ただしその際には出席費を支払い、申請書を提出しなければならない。
また、TEDの大切な目的の一つとして、団体理念を広めるために各地域コミュニティで同じように講演会が行われることをサポートしている。

しかし設立当初の1984年にはここまで知名度が高く、また広く世界に公に公開されていたものではない。当時は限定された範囲のみの公開だった。
そこから2001年に現在に続くクリスアンダーソンがTEDのキュレーターとなり、より多くの言語、コンテンツ、アイディアが広められた。
そして2001年から2006年にかけてTEDは大きな変革の時期でもあった。変革とは、毎年北アメリカで行われているTEDカンファレンスとは異なり世界中で行われるTEDGlobalが開催される、TED賞の授与が開始する、TEDカンファレンスのコンテンツがTEDTalksとしてオンライン上に公開されるようになった。
この時公開された最初の6人のTEDTalksはすでに100万以上の視聴回数に登り、2007年にはこの知名度ゆえにTEDの公式ウェブサイトが再度公開され、より多くの人が無料で動画を見ることができるようになった。
その後2008年には春に開催されるTEDカンファレンス自体により多くの人がより低価格で参加できるようになったTEDActiveが開催された。続く2009年にはTEDTalksの視聴回数は1億以上に登った。
同じく2009年に、新進気鋭のイノベーターを世界中から無料で招待させるというTED Fellows Program が開催され、TEDカンファレンスへの参加自体もより世界中に開けたものとなった。同じ年にはTEDxが誕生しTEDの理念を受け継いだ地域の独立プログラムが多く開催されることとなった。同じ頃、TED Translatorプログラムに着手し、TED Talkが100以上の言語に翻訳されるための基盤が作られた。
2012年3月には、TED-Edが開始され、教育者向けの短いビデオレッスンが作成された。また、2012年4月には、TED Talkからのアイデアや話を公共のラジオリスナーにもたらすパートナーシップであるTED Radio Hourが配信されるようになった。無料でアクセスを拡大することでより多くの人にアイディアを拡散する取り組みが次々に開催された。
2012年秋、とうとうTED Talksは視聴回数が10億回を超えた。毎秒平均17ページの新しいページビューで、TEDトークが世界中で見られ続けているので、TEDカンファレンスやイベントは参加者を刺激し、やる気を起こさせ続けている。2014年には、毎年恒例のTEDカンファレンスがカナダのバンクーバーで30周年を迎えた。

上記したように様々な歴史が連なり、今となってはTEDカンファレンスは世界的に知名度は広まり、過去にはマイクロソフト創業者ビルゲイツ、アップルの生みの親スティーブ・ジョブズ等著名人も数多く公演している。
普通の講演会との一番の違いは、講演者の分野が様々であり、どんな分野の人でも誰もがその場で平等にスピーチをすることができる点である。
そして同時にアーカイブ映像を通じて、世界中の誰もが、様々な垣根を超えてアイディアを共有することができ、その影響力が大きいという点で一線を画している。
そして、このエンターテイメント性、手軽さゆえに、TEDの動画は今日本国内でも多くの注目を集める。それは決して社会人だけでなく、中高校生にも及び、英語の受験勉強にも活用されている。

またTEDはライセンスに関して非営利目的であれば Creative Commons License に基づいている。Creative Commons License とはインターネット時代のための新しい著作権ルールで、作品を公開する作者が「この条件を守れば私の作品を自由に使って構いません。」という意思表示をするためのツールである。利用することで、作者は著作権を保持したまま作品を自由に流通させることができ、受け手はライセンス条件の範囲内で再配布やリミックスなどをすることができる。TEDの場合はCC BY – NC – ND 4.0 International、クレジットの表記ー非営利目的下であることー改変禁止、を義務づけている。

#### 第2章 2.TEDxとは<a name="2.TEDxとは"></a>

TEDxとは、TEDのIdea worth spreading の精神を受け継ぎ、世界各国、各コミュニティから発信する独自のアイディアを発信するものだ。
実際のTEDxイベントではその一環としてそれぞれのコミュニティごとに登壇するライブスピーカーのスピーチ、またTEDカンファレンス等のスピーチが配信されているTEDTalksの中から動画を選択し流さなければならない。
それぞれのTEDxコミュニティがTEDから正式にライセンスを取得し、TED本国とはまた独立した団体として、それぞれのアイディアの発信場所としてカンファレンスを開催する。

そして本国のTEDの講演同様、このTEDxの講演も動画アーカイブとしてネット上にアップされる。
TEDxの特徴は、TEDと同様様々な分野のユニークなアイディアを発信する場所としてTEDの精神を受け継ぐ一方で、TEDとは独立した団体として各コミュニティの独自の色を出すことができる点である。バラエティ豊かな問題提起や、バラエティ豊かなジャンルのアイディアがイベントの中では提議されるのである。
また、商業的、政治的、宗教的なグループは存在せず、全てのコミュニティは非営利なボランティア団体によって形成されている。
そして、参加者、スピーカー、スタッフの垣根を関係なく、会場にいる全ての人が交流を深めることができる。
その一貫として、講演終了時には誰しも参加することができるレセプションパーティを必ず開催しなければならない。
TEDxの日本国内に置ける歴史は2009年には米国以外で初となるTEDxイベント、”TEDxTokyo”が開催されたことから始まった。それ以降は日本国内を含む世界各地へ広まり、近年ではそのブランド価値、独自性から尚一層注目が高まり、世界130カ国以上、日本では過去から400近くのTEDxイベントが開催されています。

そしてTEDxイベントは以下の種類に分けることができる。
* スタンダードイベント

一番標準的なイベントである。
* ユニバーシティイベント

各大学で開催されるイベント。スタッフも大学生、または学生であることが求められる。名称を変更すれば違うTEDxUniversity団体として一つの大学から複数のライセンスを申請することも可能。
例　TEDxKeioUSFC、TEDxHiyoshi
* ユースイベント

学校団体や、若者、子供達によって形成されるイベント
* エデュケイションイベント

教育者たちや生徒たち、が未来の教育について集まり議論するイベント
* サロンイベント

TEDxコミュニティが年に一度行う大きなTEDxイベントの他に週ごとや、月ごとに行う小さなイベント、ただしこのイベントを行うためには他イベントと同様にTEDxライセンスを持っていなければならない
* ライブイベント

毎年行われるTEDカンファレンスをサイマルキャスト上映するイベント
* ウーマンイベント

毎年解されるTEDWomanイベントのサイマルキャスト上映を中心に行うイベント、特に女性に注目するイベント

など様々な種類のイベントが存在する。
またそれぞれのイベントには各TEDxルールが適用される。
中でも本研究をするうえで開催されたTEDxAoyamaGakuinUが所属するのはTEDxUniversityイベントである。
このTEDxUniversityイベントは通常のスタンダートライセンスと基本は同じだが、その名称は各大学に帰属するため、開催に至ってはその大学の許可が必要である。

大学イベントなので、ライセンス保有者は現役の学生、教職員でなければならず、卒業生やもともとのスタッフだからと行ってライセンスの申請は行うことはできない。
ただし観客もその大学の人でなければならないというルールはなく、誰でも観覧は可能だが、最終的な決定やターゲットはイベント運営者によるのだ。
また重要なこととして、通常のTEDxUniversityイベントは会場の規模が100名までと決まっていることだ。公式のTEDカンファレンスに参加したことがある人のみ100名を超える規模のイベントを開催することができる。
また各イベントにはそのイベントのテーマを設ける必要がある。
イベント自体は多くは一日かけてイベントを行う大学が多い。

#### 第2章 3.TEDxAoyamaGakuinUとは<a name="3.TEDxAoyamaGakuinUとは"></a>

上記したTEDxイベントの種類の中のTEDxUniversityイベントの一つである。その青山学院大学バージョンとしてTEDxAoyamaGakuinUという名称を用いている。

始まりは2015年に開催されたTEDxAGUという名称のイベントであった。2015年当時は青谷学院大学ESSの中から有志の人で開催された。
青山学院大学アスタジオで開催され、およそ80名近くの参加者が参加する大きなイベントとして成功した。
しかしその後ESS内の中でライセンスを引き継ぐ人が途絶え、ライセンスは失効し、イベント自体も途絶えた。
そこから2017年に再び活動を再開した。この時はESSの主体ではなく、完全に独立した団体としてTEDxAoyamaGakuinUという学生団体として立ち上げメンバーを募集した。またライセンス自体の名称も変更し、TEDxAGUからTEDxAoyamaGakuinUとなった。
しかし近年のTEDxイベントの日本国内の乱立によりTEDxイベントのライセンス自体が取得困難となりTEDxAoyamaGakuinUの開催は危ぶまれた。
2018年3月にライセンスの申請は許可され、このライセンスが失効しない1年のうちにTEDxAoyamaGakuinUを開催することが決まった。

開催理由として、2015年から時を経て今現在、陸上の連勝記録やシンギュラリティラボの設置でまた新たに注目を浴びている青山学院大学から、青山学院大学のOBOGの方々の素晴らしいアイディアや、教授や関わる人々、学生たちのユニークなアイディアを発信する場所が必要であり、そう言った人たちが互いに交流を深めさらにそれを聞いた人々との交流を深めることで、より深いアイディアが青山学院大学から発信される可能性を高めるという目的があった。
ここから参加者のターゲットも青山学院大学の学生や教員やOBOGを中心としているが、広く一般的に募集した。

そして各大学イベントにはテーマが存在する。TEDxAoyamaGakuinU2018年イベントにおいては「unlearn+」（アンラーンプルス）というテーマでイベントを行った。
意味は、日本語で一言で表すと「学びほぐす」という。この言葉には、すでにある知識や情報だけで満足することなく、TEDxイベントを利用してあらゆる分野の人の話を聞くという経験を通し、より自分の考えを深めていくという思いが込められている。さらに「+」（プルス）には「もっと先へ」という意味が込められており、さらなる価値観の広がりなどを表す。
