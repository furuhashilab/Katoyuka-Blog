### 第1章　はじめに<a name="はじめに"></a>

本研究は、TEDxAoyamaGakuinU 2018というソーシャルイベントの一つにおけるレセプション/アフターパーティでの孤立問題を改善する取り組みとして、イベントの中に様々試作を盛り込んだ方法と結果、その考察である。TEDxに限らず、不特定多数が交流するソーシャルイベント後のアフターパーティでは初対面同士の会話がしづらく、孤立してしまう通称「レセプション難民」現象が多々発生している。せっかく足を運んでいただいた人々が少しでも有意義な時間を過ごしていただくために、この「レセプション難民」問題を減らすいくつかのアイディアの効果検証を今回のTEDxAoyamaGakuinUでは実施した。

背景として、様々なスピーチ動画を配信する非営利団体TEDはよく知られた名前である。そこからまた独立した団体として、各地域コミュニティにTEDx団体が存在する。近年このTEDxイベントを行う団体は数を増し、1番始めに行われたTEDxTokyoから日本国内だけで400近くにまで増えた。

その中で私は自身の所属する青山学院大学でTEDxAoyamaGakuinUとしてイベントを開催することを決定した。イベント開催のために様々な準備をしてきた中で、他団体のTEDxの運営を手伝うために、また単純に観覧するためにオーディエンスとして参加することで自分の中のTEDxの知見を高めようとした。
参加することで感じた課題がTEDx運営上のレセプション会場の孤立問題である。自分自身がレセプションに参加することで初対面の人との会話が困難であり、また他者との会話の中で同じように会話にハードルを感じる人が存在するとわかった。

このような想いから自身が主催するTEDxAoyamaGakuinUではこうした「レセプション難民」を減少させようと取り組んだ。

また、この取り組みの中で、数多く存在する他のTEDxとの差異を出すために、またレセプションの参加者をより楽しませるために、策を考え実行し考察した。
この研究が来年以降のイベントでのレセプションへ貢献し、TEDxAoyamaGakuinUがより広く大勢の人々に楽しんでもらえるようなイベントになることを期待する。

本論文の以下の構成は次のようになっている。
 
 第２章では、本論文で使用する諸概念について述べる。
 
 第３章では、レセプション時に用いた検証施策について述べる。
 
 第４章では、それぞれの施策の検証結果、考察を述べる。
 
 最後に、第５章で本論文の結論を述べる。

### 第2章　テーマ概論説明
#### <a name="第2章　1.TEDとは"></a>

TEDとは -Idea worth spreading- とあるように、価値有るアイディアを世に広めることを目的として非営利団体である。1984年に始まった当初、TEDの名前のそれぞれの頭文字の通り、T... Technology (テクノロジー) E… Entertainment (エンターテイメント) D… Design (デザイン)に集中したおよそ18分前後にわたる短く、かつ印象的な動画を配信していた。今日ではジャンルに関係なく科学やビジネス、国際問題など様々なトピックに関するスピーチ動画を配信している。

現在の動画数は2900以上、公開されている動画の言語数は100以上に上る。
そしてこの団体は国際的コミュニティであり、価値あるアイディアを世に広めるという目的においてどんな人にも広く公開されている。本場の講演会にスピーカーとして参加することも可能である。ただしその際には出席費を支払い、申請書を提出しなければならない。
また、TEDの大切な目的の一つとして、団体理念を広めるために各地域コミュニティで同じように講演会が行われることをサポートしている。

しかし設立当初の1984年にはここまで知名度が高く、また広く世界に公に公開されていたものではない。当時は限定された範囲のみの公開だった。
そこから2001年に現在に続くクリスアンダーソンがTEDのキュレーターとなり、より多くの言語、コンテンツ、アイディアが広められた。
そして2001年から2006年にかけてTEDは大きな変革の時期でもあった。変革とは、毎年北アメリカで行われているTEDカンファレンスとは異なり世界中で行われるTEDGlobalが開催される、TED賞の授与が開始する、TEDカンファレンスのコンテンツがTEDTalksとしてオンライン上に公開されるようになった。
この時公開された最初の6人のTEDTalksはすでに100万以上の視聴回数に登り、2007年にはこの知名度ゆえにTEDの公式ウェブサイトが再度公開され、より多くの人が無料で動画を見ることができるようになった。
その後2008年には春に開催されるTEDカンファレンス自体により多くの人がより低価格で参加できるようになったTEDActiveが開催された。続く2009年にはTEDTalksの視聴回数は1億以上に登った。
同じく2009年に、新進気鋭のイノベーターを世界中から無料で招待させるというTED Fellows Program が開催され、TEDカンファレンスへの参加自体もより世界中に開けたものとなった。同じ年にはTEDxが誕生しTEDの理念を受け継いだ地域の独立プログラムが多く開催されることとなった。同じ頃、TED Translatorプログラムに着手し、TED Talkが100以上の言語に翻訳されるための基盤が作られた。
2012年3月には、TED-Edが開始され、教育者向けの短いビデオレッスンが作成された。また、2012年4月には、TED Talkからのアイデアや話を公共のラジオリスナーにもたらすパートナーシップであるTED Radio Hourが配信されるようになった。無料でアクセスを拡大することでより多くの人にアイディアを拡散する取り組みが次々に開催された。
2012年秋、とうとうTED Talksは視聴回数が10億回を超えた。毎秒平均17ページの新しいページビューで、TEDトークが世界中で見られ続けているので、TEDカンファレンスやイベントは参加者を刺激し、やる気を起こさせ続けている。2014年には、毎年恒例のTEDカンファレンスがカナダのバンクーバーで30周年を迎えた。

上記したように様々な歴史が連なり、今となってはTEDカンファレンスは世界的に知名度は広まり、過去にはマイクロソフト創業者ビルゲイツ、アップルの生みの親スティーブ・ジョブズ等著名人も数多く公演している。
普通の講演会との一番の違いは、講演者の分野が様々であり、どんな分野の人でも誰もがその場で平等にスピーチをすることができる点である。
そして同時にアーカイブ映像を通じて、世界中の誰もが、様々な垣根を超えてアイディアを共有することができ、その影響力が大きいという点で一線を画している。
そして、このエンターテイメント性、手軽さゆえに、TEDの動画は今日本国内でも多くの注目を集める。それは決して社会人だけでなく、中高校生にも及び、英語の受験勉強にも活用されている。
